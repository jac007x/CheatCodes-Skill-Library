❓ New here? What is a skill and how do I use it?

A skill is a ready-made prompt — a set of instructions you paste into an AI assistant like ChatGPT, Claude, or Microsoft Copilot. The skill tells the AI exactly what to do, so you get a great result without figuring out the right words yourself.

  1. Click any skill card below.
  2. Click "🚀 Use this skill" and fill in a few details about your situation.
  3. Click "Copy prompt".
  4. Paste it into your AI assistant and read the result. Done!

Prefer the terminal? Run the interactive concierge from your command line: python scripts/concierge.py
