🎯 PROPER ENGINE SPECIFICATIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

What ProperPhase2Analyzer Does:

1. TREND ANALYSIS
   ├─ Direction (up/down/flat)
   ├─ Momentum (accelerating/stable/decelerating)
   ├─ Volatility measurement
   ├─ Consistency scoring
   └─ Next-period forecasting

2. DRIVER ANALYSIS
   ├─ Primary driver identification
   ├─ Contribution analysis (% of total)
   ├─ Growth rate per driver
   ├─ Secondary driver ranking
   └─ Drag identification (negative contributors)

3. ANOMALY DETECTION
   ├─ Statistical spike/drop detection
   ├─ Magnitude scoring (standard deviations)
   ├─ Likely cause inference
   │  ├─ Holiday/seasonal
   │  ├─ Promotion period
   │  ├─ External event
   │  └─ Quality issue
   └─ Action recommendations per type

4. IMPACT SCORING
   ├─ Business impact (1-5 scale)
   ├─ Investor impact (1-5 scale)
   ├─ Urgency level (1-5 scale)
   ├─ Confidence score (0-1)
   ├─ Overall sentiment classification
   └─ Priority ranking

5. NARRATIVE GENERATION
   ├─ Multiple styles:
   │  ├─ Executive (concise, bottom-line)
   │  ├─ Investor (growth-focused)
   │  ├─ Technical (analytical)
   │  ├─ Board (strategic)
   │  ├─ Cautious (risk-aware)
   │  └─ Optimistic (opportunity-focused)
   ├─ Template-based generation
   ├─ Or AI-powered via Element LLM Gateway
   └─ Automatic style selection

6. SMART RECOMMENDATIONS
   ├─ Data-driven and specific
   ├─ Actionable (allocate, expand, investigate)
   ├─ Prioritized by urgency
   ├─ Different recommendations per segment
   ├─ Success criteria implied
   └─ Owner/function assignment ready

💡 KEY INSIGHT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Current Phase2Analyzer is a "metric summary engine" that tells WHAT happened.

Proper insight engine is a "narrative intelligence engine" that tells:
  1. WHAT happened (metrics)
  2. WHY it happened (drivers)
  3. IS IT NORMAL (anomalies)
  4. DOES IT MATTER (impact score)
  5. WHAT TO DO (recommendations)
  6. WHAT'S NEXT (forecast)

This is the difference between data and insight.
